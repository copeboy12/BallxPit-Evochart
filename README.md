# BALL x PIT Evolution Chart

A static GitHub Pages-ready website inspired by the VSEVO chart layout, rebuilt for BALL x PIT.

The public website files are in `docs/`:

- `docs/index.html`
- `docs/styles.css`
- `docs/script.js`
- `docs/.nojekyll`

## Publish With GitHub Pages

1. Create a public GitHub repository.
2. Push this project to the repository.
3. In the repository, open `Settings > Pages`.
4. Set the source to `Deploy from a branch`.
5. Pick branch `main` and folder `/docs`.
6. Save.

Your site will publish at:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```

GitHub Pages can publish static HTML, CSS, and JavaScript directly from a repository branch.
